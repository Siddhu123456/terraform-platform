# Release Notes

## Purpose

This directory contains the official release notes for the AI-Native Terraform Platform.

Release notes summarize the changes introduced in each published version of the platform.

---

## Objectives

Release notes provide:

* Release summary.
* New features.
* Improvements.
* Bug fixes.
* Breaking changes.
* Compatibility information.

---

## Naming Convention

Release note files shall follow semantic versioning.

Examples:

```text
v1.0.0.md

v1.1.0.md

v2.0.0.md
```

---

## Repository Policy

A release note shall be created for every published platform release.

Release notes form part of the permanent repository history.
